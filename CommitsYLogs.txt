/* Commits */
Editar último commit no remoto: git commit --amend

Borrar último commit no remoto: git reset --hard HEAD^
	Sincronizar remote: git push origin -f
Borrar un commit: git revert (commitName)
Borrar cambios sin commit: git reset --hard
Descartar un cambio commiteado: git reset HEAD CONTRIBUTING.md
Descartar cambios commiteados: git reset HEAD
Descartar cambios totalmente: git checkout -- .
	git clean -d -f

/* Logs */
últimos 2 logs: git log -p -2
Ver log en un rango: git log --since=2.weeks
Archivos y lineas modificadas en el log: git log --stat
Logs en linea: git log --pretty=oneline
Log en formato: git log --pretty=format:"%h - %an, %ar : %s"
										a82a6d - Scott Chacon, 6 years ago : Change v
	formatos:
		%H: Commit hash
		%h: Abbreviated commit hash
		%T: Tree hash
		%t: Abbreviated tree hash
		%P: Parent hashes
		%p: Abbreviated parent hashes
		%an: Author name:
		%ae: Author email
		%ad: Author date (format respects the --date=option)
		%ar: Author date, relative
		%cn: Committer name
		%ce: Committer email
		%cd: Committer date
		%cr: Committer date, relative
		%s: Subject
